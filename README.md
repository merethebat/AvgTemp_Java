# Average Temperature Homework
## Merethe Batino
----

## Description
This program asks the user to enter temperature values for a certain number of days.
After all values are entered, the program calculates the average temperature and shows how many days were above the average.

## Features
User inputs the number of days
Stores temperature values in an array
Calculates the average temperature
Counts temperatures above the average

## Language Used
Java

## Example
How many days of temperature? 5
Enter temperature for day 1: 20
Enter temperature for day 2: 25
Enter temperature for day 3: 18
Enter temperature for day 4: 30
Enter temperature for day 5: 22

Average temperature: 23.0
Days above average: 2
